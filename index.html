PROJETOS

DATAHANDLER

Uma aplicação desenvolvida no Google Colab, utiliza a API do Google Gemini para classificar dados de exames de mercadorias pré-embaladas por segmento. Mercadorias pré-embaladas são alvo de fiscalização do INMETRO quanto ao conteúdo efetivo (massa, volume, comprimento, etc.)

VALIDATE REPORTS

Uma aplicação desenvolvida no Google Colab com a qual laudos de exames de mercadorias pré-embaladas são avaliados quanto à gravidade de erros constatados. A depender da gravidade, um documento é gerado com recomendações previstas no Regulamento Técnico do INMETRO

Desafio DIO Responsividade - Clone Discord

Desafio de projeto relacionado à trilha CSS/Responsividade do curso BootCamp Frontend DIO Santander. Trata-se do clone de uma página do Discord, utilizando HTML e CSS.
